# Task Manager App

A simple Task Management web application built using Spring Boot.  
This project allows users to create, view, update, and delete tasks.

---

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- HTML (Frontend)
- MySQL / H2 Database
- Maven

---

## ✨ Features
- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Simple UI using HTML

---

## 📁 Project Structure
- Controller → Handles HTTP requests
- Model → Task entity
- Repository → Database operations
- Resources → HTML, SQL, configuration files
